# Arduino-Assignment-2
Arduino project- beeping countdown using tone() and 7 segment display 
This is my Assignment 2 for programming C++ using Arduino at Ghana Communication Technology University.


//what the ptoject does//
A 7 segment display which counts down from 9 to 0. On each step a passive buzzer plays a short beep. When the countdown reaches 0, tye buzzer plays a longer tone to signal completion. 

//hardware used//
1 passive buzzer 
1 single digit 7-segment display
1 220 ohm resistor 
breadboard and jumper wires 

//concepts demonstrated//
the tone() and notone() functions 
passive vs active buzzers
7- segment diplay wires 
2D arrays for digit patterns 
functions with parameters
while and for loops 
serial monitor outputs

//how to run//
wire the buzzer to pin 8 
wire the 7- segments a- g to Arduino pins 2,3,4,5,6,7,8,9
connect the COM pin of the display to GND via a 220 o resistor
open the .ino file in the Arduino IDE 
select tools > board > Arduino-Uno and tge correct port 
click upload
